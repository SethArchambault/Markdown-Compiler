# Markdown-Compiler
A minimal markdown compiler for my personal website, written in C in a Handmade style.

# Setup

Create this folder structure:

```
mkdir my_website
cd my_website
mkdir public
mkdir html_generator
git clone git@github.com:SethArchambault/Markdown-Compiler.git
```
